# JQuery介绍

## JQuery重要特性

+ JavaScript特效
+ AJAX
+ DOM



##下载安装JQuery

​	[jquery](<https://jquery.com/download/>)

​	然后放到待使用页面相同的目录中



## 在线使用JQuery

使用Google CDN

```java
<head>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js">
</script>
</head>
```



## 添加使用JQuery:

```java
<head>
<script type="text/javascript" src="jquery.js"></script>
</head>
```
